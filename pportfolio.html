Intelli-Invest

Overview

IntelliInvest is an advanced financial project designed to revolutionize the way investment decisions are made by integrating traditional financial analysis with cutting-edge technologies. This project combines technical indicators, machine learning for time series prediction, and portfolio organization to create a sophisticated tool for investors. Additionally, web development integration allows for dynamic querying and response generation using large language models (LLMs), providing users with human-like interaction and insightful financial advice.

Key Components

1.Traditional Financial Analysis

In-depth analysis of financial statements, ratios, and other fundamental metrics. Evaluation of a companyâ€™s financial health and performance using conventional methods.

2.Technical Indicators

Incorporation of various technical indicators such as moving averages, Relative Strength Index (RSI), and Bollinger Bands. Understanding the underlying parameters of these indicators and analyzing how changes in parameters affect trading signals.

3.Machine Learning for Time Series Prediction

Employing machine learning algorithms to predict stock prices and market trends based on historical data. Time series analysis to identify patterns and forecast future movements in stock prices.

4.Portfolio Organization:

Application of portfolio theories including Modern Portfolio Theory (MPT) and the Efficient Frontier to optimize investment portfolios. Balancing risk and return to create diversified portfolios that meet specific investment goals.

5.Web Development and LLM Integration

Development of a user-friendly web interface for seamless interaction with the IntelliInvest system. Integration of large language models (LLMs) to generate human-like responses to user queries, providing personalized investment advice and insights.

Virtual Reality Using Blender

Overview

The "Virtual Reality Using Blender" project helped me to navigate through the essentials of creating your own immersive virtual reality (VR) environment using Blender, a powerful open-source 3D creation suite. This project gave me hands-on experience in 3D modeling and rendering, allowing me to bring virtual worlds to life.

Key Components

1.3D Modelling

Learnt fundamentals of 3D modeling, including the creation of various objects and structures within Blender. Developed skills in manipulating vertices, edges, and faces to create detailed and accurate models.

2.Texturing

Gained expertise in applying textures to the 3D models to add realism and detail. Explored UV mapping techniques to ensure textures are properly aligned and look natural on the models.

3.Lighting

Understood the principles of lighting within a 3D environment. Experimented with different types of lights and learnt how to position them to create the desired mood and atmosphere in the virtual world.

4.Rendering

Mastered the rendering process to produce high-quality images and animations of 3D scenes. Learnt about various rendering settings and how to optimize them for the best performance and visual quality.
